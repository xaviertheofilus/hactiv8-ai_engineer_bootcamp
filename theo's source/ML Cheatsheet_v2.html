Decision Tree
DecisionTreeClassifier / DecisionTreeRegressor
Both
Urutan Tuning ↓
★★★
max_depth
Kedalaman maksimum pohon. Parameter paling krusial.
↑Overfit ↓Underfit
★★★
min_samples_split
Min sampel untuk split node. Default=2 (sangat mudah overfit).
↑Underfit ↓Overfit
★★
min_samples_leaf
Min sampel di leaf node. Mirip min_samples_split tapi lebih kuat.
↑Underfit ↓Overfit
★★
max_features
Jumlah fitur yang dipertimbangkan tiap split. "sqrt" untuk clf.
↑Overfit ↓Variance ↓
★
max_leaf_nodes
Batas jumlah leaf. Alternatif kontrol kompleksitas vs max_depth.
↑Overfit ↓Underfit
💡 Praktik: Mulai dari max_depth=3~5. Kalau masih underfit, longgarkan min_samples_split. Jangan biarkan default — DT sangat cepat overfit.
python

Random Forest
RandomForestClassifier / RandomForestRegressor
Both
Urutan Tuning ↓
★★★
n_estimators
Jumlah pohon. Lebih banyak = lebih stabil, tapi lambat. Minimal 100.
↑Variance ↓ ↓Instabil
★★★
max_depth
Kedalaman tiap pohon. RF lebih toleran overfit tapi tetap perlu dijaga.
↑Overfit ↓Underfit
★★★
max_features
"sqrt" (clf) / "1/3" (reg). Kontrol diversitas antar pohon.
↑Overfit ↓Bias ↑
★★
min_samples_leaf
Min sampel di leaf. Naik ke 4-10 untuk dataset kecil/noisy.
↑Underfit ↓Overfit
★
bootstrap / max_samples
Proporsi data untuk tiap pohon. Kurangi untuk lebih random.
↑Overfit ↓Diversitas ↑
💡 Praktik: Set n_estimators=200+ dulu. Lalu tuning max_features & min_samples_leaf. RF biasanya out-of-the-box bagus, tidak perlu tuning ekstrem.
python

Gradient Boosting / XGBoost
GradientBoostingClassifier / XGBClassifier / LightGBM
Both
Urutan Tuning ↓
★★★
learning_rate (eta)
Kontrol kontribusi tiap pohon. Kecil = perlu n_estimators lebih banyak.
↑Overfit ↓Underfit
★★★
n_estimators
Jumlah boosting rounds. Harus seimbang dengan learning_rate.
↑Overfit ↓Underfit
★★★
max_depth
GBM sensitif depth. Biasanya cukup 3-6. XGB default=6.
↑Overfit ↓Underfit
★★
subsample
Proporsi data per pohon (stochastic GBM). 0.5-0.8 bagus.
↑→1.0Overfit ↓Regularisasi
★★
reg_alpha / reg_lambda
L1 (alpha) dan L2 (lambda) regularisasi. Default lambda=1.
↑Underfit ↓→0Overfit
★★
colsample_bytree
Fraksi fitur per pohon. 0.5-0.8 untuk regularisasi fitur.
↑→1.0Overfit ↓Bias ↑
★
min_child_weight
Min sum of instance weight in child. Mirip min_samples_leaf.
↑Underfit ↓Overfit
💡 Praktik: Mulai lr=0.1, cari n_estimators optimal dengan early_stopping. Lalu turunkan lr=0.01-0.05 dan naikkan n_estimators. Gunakan GridSearch/Optuna untuk max_depth + subsample.

Support Vector Machine
SVC / SVR
Both
Urutan Tuning ↓
★★★
C
Regularisasi. Nilai besar = fit data lebih ketat (penalty misclassify besar).
↑Overfit ↓Underfit
★★★
kernel
Tipe kernel: linear / rbf / poly / sigmoid. RBF paling umum.
rbf/polyNon-linear linearSimple
★★★
gamma
Jangkauan pengaruh satu sampel (untuk rbf/poly). "scale" = 1/(n_feat×var).
↑Overfit ↓Underfit
★★
degree (poly kernel)
Derajat polynomial. Hanya relevan jika kernel="poly".
↑Overfit ↓Underfit
★
epsilon (SVR only)
Lebar "tube" toleransi error. Besar = lebih toleran, model lebih smooth.
↑Underfit ↓Overfit
💡 Praktik: SVM wajib feature scaling dulu! GridSearch pada C=[0.01, 0.1, 1, 10, 100] dan gamma=['scale', 0.001, 0.01, 0.1]. Lambat untuk dataset besar (>10k baris).
python
from sklearn.svm import SVC, SVR
from sklearn.model_selection import GridSearchCV
from sklearn.pipeline import Pipeline
from sklearn.compose import ColumnTransformer
from sklearn.impute import SimpleImputer
from sklearn.preprocessing import StandardScaler, OneHotEncoder

# Column definitions — sesuaikan dengan dataset
num_cols = [num1, num2]  # numeric columns
cat_cols = [cat1, cat2]  # categorical columns

# Numeric transformer — imputer → scaling
numeric_transformer = Pipeline([
    (imputer, SimpleImputer(strategy=median)),
    (scaler,  StandardScaler())
])

# Categorical transformer
categorical_transformer = Pipeline([
    (encoder, OneHotEncoder(handle_unknown=ignore))
])

# Preprocessor
preprocessor = ColumnTransformer(transformers=[
    (num, numeric_transformer, num_cols),
    (cat, categorical_transformer, cat_cols)
])

# Classifier
classifier = SVC(C=1.0, kernel=rbf, gamma=scale, probability=True)

# Pipeline — scaling aman dari leakage karena ada dalam pipeline
pipeline_svc = Pipeline([
    (preprocessor, preprocessor),
    (classifier,   classifier)
])
pipeline_svc.fit(X_train, y_train)
y_pred  = pipeline_svc.predict(X_test)
y_proba = pipeline_svc.predict_proba(X_test)[:, 1]

# Regression
regressor    = SVR(C=1.0, kernel=rbf, epsilon=0.1)
pipeline_svr = Pipeline([(preprocessor, preprocessor), (regressor, regressor)])

# Tuning
param_grid = {
    classifier__C:     [0.01, 0.1, 1, 10, 100],
    classifier__gamma: [scale, 0.001, 0.01, 0.1],
}
search = GridSearchCV(pipeline_svc, param_grid, cv=5, scoring=roc_auc, n_jobs=-1)
search.fit(X_train, y_train)
K-Nearest Neighbors
KNeighborsClassifier / KNeighborsRegressor
Both
Urutan Tuning ↓
★★★
n_neighbors (k)
Jumlah tetangga. Parameter paling kritis. Coba k=1..20.
↑Underfit ↓ (k=1)Overfit
★★
weights
"uniform" vs "distance". Distance beri bobot tetangga terdekat lebih besar.
distanceOverfit ↑ uniformGeneralize
★★
metric
Euclidean (p=2) default. Manhattan (p=1) untuk data high-dim/sparse.
~Data-dependent
★
algorithm
ball_tree / kd_tree / brute. Hanya pengaruhi speed, bukan akurasi.
~Speed only
💡 Praktik: Wajib scaling! Gunakan k ganjil untuk binary clf. Rule of thumb: k = √(n_samples). Performa KNN drop drastis di high-dimensional data (curse of dimensionality).

Inverse of regularization strength. Kecil = regularisasi kuat.

Ridge Regression (L2)
Ridge / RidgeClassifier
Regression Classification
Urutan Tuning ↓
★★★
alpha (λ)
Kekuatan L2 regularisasi. Shrink semua koefisien mendekati nol.
↑Underfit ↓→0→ OLS
★★
fit_intercept
Intercept tidak diregularisasi. Selalu True kecuali data sudah centered.
~Standard
★
solver
'auto', 'svd', 'cholesky', 'saga'. Auto pilih terbaik berdasarkan data.
~Speed only
💡 Praktik: Gunakan RidgeCV untuk auto-tune alpha. Bagus saat ada multicollinearity. Tidak nol-kan koefisien — gunakan Lasso untuk feature selection.

from sklearn.linear_model import Ridge, RidgeCV
from sklearn.model_selection import GridSearchCV
from sklearn.pipeline import Pipeline
from sklearn.compose import ColumnTransformer
from sklearn.impute import SimpleImputer
from sklearn.preprocessing import StandardScaler, OneHotEncoder

# Column definitions — sesuaikan dengan dataset
num_cols = [num1, num2]  # numeric columns
cat_cols = [cat1, cat2]  # categorical columns

# Numeric transformer — imputer → scaling
numeric_transformer = Pipeline([
    (imputer, SimpleImputer(strategy=median)),
    (scaler,  StandardScaler())
])

# Categorical transformer
categorical_transformer = Pipeline([
    (encoder, OneHotEncoder(handle_unknown=ignore))
])

# Preprocessor
preprocessor = ColumnTransformer(transformers=[
    (num, numeric_transformer, num_cols),
    (cat, categorical_transformer, cat_cols)
])

# Regressor
regressor = Ridge(alpha=1.0)

# Pipeline
pipeline_ridge = Pipeline([
    (preprocessor, preprocessor),
    (regressor,    regressor)
])
pipeline_ridge.fit(X_train, y_train)

# RidgeCV — auto-tune alpha
regressor_cv    = RidgeCV(alphas=[0.001, 0.01, 0.1, 1.0, 10, 100], cv=5)
pipeline_ridge_cv = Pipeline([(preprocessor, preprocessor), (regressor, regressor_cv)])
pipeline_ridge_cv.fit(X_train, y_train)
print(Best alpha:, pipeline_ridge_cv.named_steps[regressor].alpha_)

# Tuning via GridSearchCV
param_grid = {regressor__alpha: [0.001, 0.01, 0.1, 1, 10, 100]}
search = GridSearchCV(pipeline_ridge, param_grid, cv=5, scoring=r2)
search.fit(X_train, y_train)

classifier = RandomForestClassifier(random_state=42)

# ── 2. Baseline Pipeline ──────────────────────────────
pipeline_base = Pipeline([
    (preprocessor, preprocessor),
    (classifier,   classifier)
])
cv = StratifiedKFold(n_splits=5, shuffle=True, random_state=42)
base_scores = cross_val_score(pipeline_base, X_train, y_train, cv=cv, scoring=f1_weighted)
print(Baseline F1: {base_scores.mean():.4f} ± {base_scores.std():.4f})

# ── 3. RandomizedSearch — prefix "classifier__" ───────
param_dist_rf = {
    classifier__n_estimators:    randint(50, 500),
    classifier__max_depth:       [3, 5, 7, 10, None],
    classifier__max_features:    [sqrt, log2, 0.3, 0.5],
    classifier__min_samples_leaf: randint(1, 10),
}
rand_search = RandomizedSearchCV(
    pipeline_base, param_dist_rf,
    n_iter=80, cv=cv, scoring=f1_weighted, n_jobs=-1, random_state=42, verbose=1
)
rand_search.fit(X_train, y_train)
best = rand_search.best_params_
print(Best random params:, best)

# ── 4. GridSearch fokus ───────────────────────────────
classifier_tuned = RandomForestClassifier(random_state=42)
pipeline_tuned   = Pipeline([(preprocessor, preprocessor), (classifier, classifier_tuned)])
param_grid_rf = {
    classifier__n_estimators:    [best[classifier__n_estimators] - 50,
                                best[classifier__n_estimators],
                                best[classifier__n_estimators] + 50],
    classifier__max_depth:       [best[classifier__max_depth]],
    classifier__max_features:    [best[classifier__max_features]],
    classifier__min_samples_leaf: [max(1, best[classifier__min_samples_leaf] - 1),
                                  best[classifier__min_samples_leaf],
                                  best[classifier__min_samples_leaf] + 1],
}
grid_rf = GridSearchCV(
    pipeline_tuned, param_grid=param_grid_rf,
    verbose=3, refit=True, scoring=precision, cv=5,
    return_train_score=True, n_jobs=2
)
grid_rf.fit(X_train, y_train)

print(Best parameters:, grid_rf.best_params_)
print(Best Precision score:, grid_rf.best_score_)

# ── 5. Evaluasi final di test set (satu kali!) ────────
final_pipeline = grid_rf.best_estimator_  # Pipeline lengkap
y_pred = final_pipeline.predict(X_test)
print(classification_report(y_test, y_pred))
print(Test Score: {final_pipeline.score(X_test, y_test):.4f})

# ── Akses komponen setelah fit ────────────────────────

fitted_classifier   = final_pipeline.named_steps[classifier]
